# Reference Material

## Do's & Don'ts

**Do:**

- Leverage RxJS for async flows.
- Prefer OnPush and pure data flows.
- Encapsulate business logic in services.

**Don't:**

- Overuse `any` types.
- Heavy logic in templates.
- Global mutable singletons.
